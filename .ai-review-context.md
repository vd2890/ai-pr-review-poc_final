# Databricks AI Review Rules

This repository contains PySpark pipelines.

## Spark Best Practices

- Do NOT use collect() on large DataFrames
- Avoid toPandas() in production
- Use distributed transformations
- Avoid driver-side loops

## Performance

- Avoid large shuffles
- Use partitioning
- Cache when reused

## Code Quality

- Add unit tests
- Avoid hardcoding paths
- Handle null values

## Output expectation

Always suggest:
- better Spark patterns
- unit tests
- performance improvements
